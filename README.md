##  Project Overview
An end-to-end Power BI analysis of 5,000 employees to identify the drivers of stress and the impact of remote work on mental wellbeing.

----

##  Key Insights
* **Isolation Tipping Point:** Remote workers show a 40% higher frequency of 'High' isolation ratings compared to Hybrid peers.
* **Meeting Fatigue:** Stress levels spike significantly after a threshold of 10 virtual meetings per week.
* **Support Gap:** Despite access to mental health resources, high-stress roles (e.g., Sales, Software Engineering) show lower satisfaction levels, indicating a need for targeted intervention.

----

##  Technical Implementation
* **Data Modeling:** Optimized Star Schema with 1-to-Many relationships.
* **DAX Measures:** Custom measures for 'Avg Stress', 'Burnout Alert %', and 'Support Delta'.
* **Visualization:** Custom-designed UI with a silver/gold executive theme for high readability.
